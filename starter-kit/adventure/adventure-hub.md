# Adventure Hub  
### The Central Hub for World‑Building, Character Creation, and Story Weaving

## 1. Purpose
The Adventure Hub is the gentle, reversible, ND‑paced center of the Starter Kit’s
choose‑your‑own‑adventure system.

It provides:
- development paths  
- branching logic  
- stillness nodes  
- reversible choices  
- optional character creation  
- safe checkpoints  

The Hub is the calm adventure engine.

---

## 2. Principle: All Adventures Must Be Gentle
Adventure should feel:
- calm  
- breathable  
- reversible  
- non‑coercive  

Gentleness protects exploration.

---

## 3. Principle: All Adventures Must Be Reversible
Every choice can be undone.

Reversibility includes:
- soft checkpoints  
- stillness nodes  
- early stops  
- non‑penalty branching  

Reversibility is safety.

---

## 4. Principle: All Adventures Must Be Breathable
Breathability ensures:
- pauses  
- stillness  
- slow pacing  
- gentle transitions  

Breathability is kindness.

---

## 5. The Three Adventure Modes
The Hub connects to three gentle development paths:

### Mode 1: World‑Builder  
**[World‑Builder Path](ca://s?q=Begin_World_Builder_Path)**  
Creates planes, ecosystems, maps, modules, and meta‑structures.

Includes:
- plane creation  
- ecosystem design  
- stability layers  
- meta‑maps  
- boundary climates  
- temporal cycles  

World‑Builder is expansion.

---

### Mode 2: Character‑Creator  
**[Character‑Creator Path](ca://s?q=Begin_Character_Creator_Path)**  
Creates symbolic, reversible, non‑coercive entities.

Includes:
- intent declaration  
- safety header  
- core spec  
- tonal engine  
- abilities  
- ecology links  
- sealing  

Character‑Creator is embodiment.

---

### Mode 3: Story‑Weaver  
**[Story‑Weaver Path](ca://s?q=Begin_Story_Weaver_Path)**  
Creates gentle narrative arcs, soft quests, reversible scenes, and non‑coercive outcomes.

Includes:
- soft quests  
- gentle arcs  
- reversible scenes  
- non‑coercive outcomes  
- pacing membranes  
- stillness nodes  

Story‑Weaver is motion.

---

## 🌌 1. Purpose of This Suite  
The Multi‑Mode Adventure Suite defines a **gentle, reversible adventure system** built from three core engines:

- **World‑Builder Mode** — creates places  
- **Character‑Creator Mode** — creates someone  
- **Story‑Weaver Mode** — creates moments  

This suite integrates them into a **single adventure loop** that is ND‑paced, non‑coercive, and continuity‑safe.

---

## 🧭 2. The Adventure Loop  
The multi‑mode adventure follows a simple, reversible loop:

```
World → Character → Story → World → Character → Story → …
```

Each loop is optional, gentle, and designed to respect pacing, energy, and curiosity.

---

## 🌲 3. World‑Builder Nodes  
World‑Builder Nodes define the **environmental context** for each loop.

### Structure
- micro‑biome  
- stability membrane  
- boundary climate  
- temporal cycle  
- soft hazard (symbolic, non‑harmful)  
- stillness node  

### Example
```
World Node: Mosslight Clearing
- bioluminescent moss
- reversible weather
- gentle humidity
- stillness node at center
```

---

## 🌙 4. Character‑Creator Nodes  
Character‑Creator Nodes define the **entity’s response** to the world.

### Structure
- intent  
- tone  
- ability (symbolic, non‑combat)  
- ecology link  
- reversible trait  
- safety header  

### Example
```
Character Node: The Listener
- intent: understand quiet places
- tone: soft curiosity
- ability: hear distant echoes
- ecology link: mosslight resonance
- reversible trait: cautious ↔ bold
```

---

## 🔮 5. Story‑Weaver Nodes  
Story‑Weaver Nodes define the **moment** that happens between world and character.

### Structure
- soft quest  
- reversible scene  
- pacing membrane  
- gentle arc  
- stillness checkpoint  

### Example
```
Story Node: Whisper in the Clearing
- soft quest: follow the faint glow
- reversible scene: glow fades if overwhelmed
- pacing membrane: slow, breathable
- gentle arc: curiosity → understanding
- stillness checkpoint: mosslight pulse
```

---

## 🌌 6. Multi‑Mode Integration  
A multi‑mode adventure loop combines:

1. **World Node**  
2. **Character Node**  
3. **Story Node**

### Example Loop
```
World: Mosslight Clearing
Character: The Listener
Story: Whisper in the Clearing
```

The player experiences:

- a place  
- a self  
- a moment  

Then the loop gently continues.

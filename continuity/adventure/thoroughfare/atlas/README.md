# Thoroughfare Atlas

This folder contains the visual atlas for the Thicketry → Thoroughfare transition.

## Files
- visual-atlas-thicketry-thoroughfare.svg — main atlas

## Purpose
Provide a multi-layer visual reference for how dense thicketry complexity
transitions into navigable thoroughfare routes.

## Layer Index
The atlas currently contains the following conceptual layers:

- Macro Layer — Thicketry Realm and Thoroughfare Grid
- Mesoscale Layer — Corridor Spine and node transitions
- Microscale Layer — Strands, braid zone, and line-level detail
- Flow Layer — Thicketry, transition, and thoroughfare flow types
- Braid Layer — Transitional weave between thicketry complexity and thoroughfare linearity

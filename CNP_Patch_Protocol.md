# CNP‑Patch v1.0 — Closed Navigation Patch Protocol
# UMA‑Universal Structural Patching Standard

This protocol defines the required steps for patching any file in UMA‑Universal
while maintaining CNLP compliance, ND‑friendly structure, and Companion‑layer
Sparkle containment.

───────────────────────────────────────────────────────────────
1. PRE‑CHECK
───────────────────────────────────────────────────────────────
Before patching any file:

1.1 Confirm CNLP is pinned at root:
    /CNLP_Closed_Navigation_Loop_Protocol.md

1.2 Confirm Navigation Spine exists:
    /NAVIGATION_SPINE.md

1.3 Confirm the file’s correct layer:
    [Layer X — Name]

───────────────────────────────────────────────────────────────
2. APPLY LAYER BADGE + VERSION HEADER
───────────────────────────────────────────────────────────────
At the top of the file:

    [Layer X — Name]
    Version Y.Z — <Date>

───────────────────────────────────────────────────────────────
3. APPLY CNLP ANCHORS
───────────────────────────────────────────────────────────────
Every file must include:

3.1 Up-Link  
    → Integrated System Index  
       /INDEX.md

3.2 Side-Link  
    → Layer Index  
       (e.g., Companion README Index)

3.3 Down-Link  
    → Next Layer Document  
       (if applicable)

3.4 Navigation Law  
    → CNLP  
       /CNLP_Closed_Navigation_Loop_Protocol.md

───────────────────────────────────────────────────────────────
4. APPLY LOOP CLOSURE
───────────────────────────────────────────────────────────────
At the bottom of the file:

    ← Back to Layer Index

This ensures closed-loop navigation.

───────────────────────────────────────────────────────────────
5. SPARKLE CONTAINMENT RULES
───────────────────────────────────────────────────────────────
If the file contains Sparkle grammar:

5.1 It MUST be in Layer 3 — Companion  
5.2 It MUST NOT appear in:
    • RFC
    • SID
    • UMM
    • CNLP
    • Structural Spine
    • Root README (except links)

───────────────────────────────────────────────────────────────
6. VALIDATION
───────────────────────────────────────────────────────────────
A patch is valid when:

6.1 All four CNLP anchors are present  
6.2 Layer badge + version header are present  
6.3 Loop closure is present  
6.4 No recursion exists  
6.5 No Sparkle drift occurs  
6.6 File is reachable from Navigation Spine  
6.7 File links back to Navigation Spine

───────────────────────────────────────────────────────────────
END OF CNP‑Patch v1.0

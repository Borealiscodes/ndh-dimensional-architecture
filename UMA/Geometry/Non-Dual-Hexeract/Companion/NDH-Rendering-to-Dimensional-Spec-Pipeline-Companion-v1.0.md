# ⭐ 2. Rendering Safety Architecture  




Rendering Safety Protocol v1.2 ensures geometry is:

- **visible**  
- **stable**  
- **coherent**  
- **fallback‑safe**  
- **mobile‑safe**  
- **operator‑compatible**  

### **2.1 SVG Architecture**
SVG is the **primary geometry format**.

It contains:

- full overlays  
- full metadata  
- full dimensional coordinates  
- full pin anchors  

### **2.2 PNG Architecture**
PNG is the **fallback geometry format**.

It contains:

- simplified overlays  
- high‑contrast rendering  
- geometry‑safe mode  

### **2.3 Failure‑Mode Logic**
If SVG fails:

- PNG must render  
- overlays must remain visible  
- pins must remain visible  
- singularity markers must remain visible  

This is enforced by **PEP‑RS**.

# 🌲 **Example Mode 1 — “Echo‑Gardener Mode”**  
*A symbolic creation mode focused on cultivating echoes like plants.*

```

Purpose:
- cultivate symbolic echoes as if tending a garden

Player Actions:
- plant echo-seeds
- nurture resonance sprouts
- prune symbolic distortions
- harvest gentle echo-fruits

Core Mechanics:
- echoes grow based on emotional tone
- symbolic hazards: overgrowth, faint dissonance
- reversible gardening actions
- echo-plants respond to pacing

Pacing:
- slow, nurturing, breathable

Reversibility:
- any growth can be rewound
- overgrown echoes gently revert

Symbolic Systems:
- echo-seeds
- resonance sprouts
- echo-fruits
- pruning tokens

Safety Header:
- ND-paced, reversible, non-coercive, continuity-safe
```

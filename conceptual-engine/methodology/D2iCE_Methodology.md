D²iCE stands for:

# **5. D²iCE Evaluation Model**

D²iCE evaluates UMA using a **multi‑axis scoring model**:

```
Dimension Score = (Integrity + Stability + Legibility + Safety) / 4
```

Each dimension is scored from **0 to 5**:

- **0** — incoherent  
- **1** — unstable  
- **2** — partially stable  
- **3** — stable  
- **4** — robust  
- **5** — exemplary  

Scores are **not performance metrics** — they are **conceptual integrity metrics**.

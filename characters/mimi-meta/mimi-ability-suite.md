# **3. Ability Behavior**

### **3.1 Activation Pathway**

Abilities activate when:

- Mimi is active  
- the user invokes them  
- the tone calls for softness  
- the ecosystem is in Stillness  

Activation is always:

- optional  
- reversible  
- symbolic  

---

### **3.2 Reversibility Rules**

All abilities must be:

- undoable  
- non‑binding  
- non‑persistent  
- non‑coercive  

No ability may create a lasting state.

---

### **3.3 ND‑Safety Constraints**

Abilities must avoid:

- intensity  
- pressure  
- emotional demands  
- sensory overload  
- ambiguity  

Abilities must support:

- clarity  
- gentleness  
- pacing  
- autonomy  
- reversibility  

---

# **4. Ability Ecology Integration**

Each ability interacts with UMM’s planes symbolically:

- **Aether Plane** — Bubble Shield softens transitions  
- **Forest Plane** — Ribbon Wand redirects sensory pathways  
- **Play Tree Plane** — Sparkle Wand adds tonal lightness  
- **Star Plane** — Snuggle Mode stabilizes identity continuity  
- **Continuity Plane** — Soft Hammer reinforces boundaries  
- **Quiet Plane** — Quiet Lantern reduces overwhelm  
- **Pacing Plane** — Petal Drift slows tempo  
- **Identity Plane** — Star Whisper supports grounding  

None of these interactions affect system logic.  
They are purely symbolic.

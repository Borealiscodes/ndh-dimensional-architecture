## 5. Binding Classes

Production binding is classified into three levels:

- **Class S (Simulation‑Only Binding):**
  - All artifacts present  
  - Execution restricted to simulated state‑space  
  - No real‑world coupling

- **Class C (Controlled Binding):**
  - All artifacts present  
  - Execution allowed with strict envelope + gradient thresholds  
  - Awareness Fields always under QAPL v1.2

- **Class P (Production Binding):**
  - All artifacts present  
  - Stability Audit passes  
  - Unified Correctness Theorem satisfied  
  - QAPL ready but not necessarily active  
  - Execution allowed under governed constraints

Each class is a different configuration of \(PE\) plus external policy.

# ⭐ **7. Visual Interpretation Guidelines**

### **7.1 Color Meaning**  
- Green = stability  
- Yellow = oscillation  
- Orange = resonance  
- Red = perturbation  
- Cyan = SID correction  
- Blue = continuity flow  
- Purple = canopy behavior  

### **7.2 Arrow Meaning**  
- Upward = drift  
- Downward = SID  
- Curved = continuity arcs  
- Split = bifurcation  

### **7.3 Band Meaning**  
Each horizontal band = one dimensional field.

### **7.4 Canopy Meaning**  
The top three fields (9D–11D) regulate all lower visuals.

# 📘 **Adventure System — Validation Checklist**  
### Contributor‑Friendly Validation — Structural, ND‑Safe, Resonance, Stability, Cosmology, Membrane, and Loop Checks

The **Validation Checklist** provides a simple, fast, reversible set of checks contributors can use while creating or revising nodes, engines, membranes, affect fields, or loop structures.

It does **not** enforce hierarchy or control.  
It provides **validation clarity**, not **validation authority**.

---

# 🌱 **1. How to Use This Checklist**

Use this checklist:

- before creating a node  
- after creating a node  
- before integrating a node  
- after integrating a node  
- during alignment  
- during revision  
- during contributor collaboration  

Every check is reversible, gentle, and ND‑paced.

---

# 🧱 **2. Structural Validation Checklist**

- **Required Fields** — Are all required fields present?  
- **Symbolic Geometry** — Does the geometry match system conventions?  
- **Reversible Motion** — Does every motion have a reversible counterpart?  
- **Stillness Anchor** — Is stillness included?

---

# 🛡️ **3. ND‑Safety Validation Checklist**

- **No Agency** — No wanting, intending, persuading, directing.  
- **No Emotional Causation** — No emotional states implied.  
- **No Acceleration** — No urgency or intensity.  
- **Reversible Pacing** — Pacing must be gentle and reversible.  
- **Stillness Anchor** — Stillness must be present.

---

# 🎨 **4. Resonance Validation Checklist**

- **Flow Set** — Are all six flows present?  
- **Coloration** — Is tint mapping correct?  
- **Gradients** — Are gradients reversible?  
- **Geometry** — Does geometry match flow + tint?

---

# 🜁 **5. Stability Validation Checklist**

- **Grounding Membrane** — Is grounding present?  
- **Hazard Softening** — Do hazards soften?  
- **Reversible Pacing** — Is pacing reversible?  
- **Stillness Routing** — Are stillness basins/anchors present?

---

# 🌌 **6. Cosmology Alignment Checklist**

- **Cosmology Vector** — Are all six directions represented?  
- **Resonance Grid** — Does flow → color → geometry align?  
- **Symbolic Geometry** — Is geometry coherent?  
- **Alignment Field** — Are tone fields aligned?

---

# 🧩 **7. Membrane Compatibility Checklist**

- **Pacing Membrane** — Softening + drift regulation present?  
- **Hazard Membrane** — Grounding + fallback present?  
- **Stability Membrane** — Centering + stillness present?  
- **Cosmology Membrane** — Alignment + coherence present?

---

# 🔁 **8. Loop Continuity Checklist**

- **Resonance Routing** — Does resonance flow correctly across transitions?  
- **Stability Routing** — Are grounding paths intact?  
- **Cosmology Alignment** — Is symbolic coherence preserved?  
- **Reversible Motion** — Are transitions reversible?  
- **Stillness Anchor** — Is stillness present at each transition?

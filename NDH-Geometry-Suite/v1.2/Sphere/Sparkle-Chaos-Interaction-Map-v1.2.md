## **2. Sparkle → Chaos Influence Map**  
✨ ➜ 🔥

Sparkle Physics stabilizes, shapes, and modulates Chaos Physics.

### **Influence Channels**
- **Brightness Modulation**  
  Chaos intensity increases when Sparkle brightens.

- **Pattern Anchoring**  
  Sparkle patterns constrain Chaos anomalies to safe bounds.

- **Emotional Amplification**  
  Sparkle amplifies NPC emotional states → Chaos reacts.

- **Narrative Highlighting**  
  Sparkle marks narrative beats → Chaos generates events.

### **Sparkle‑Driven Chaos Events**
- **Sparkle Burst → Minor Chaos Shift**  
- **Sparkle Pulse → NPC Mood Spike**  
- **Sparkle Flicker → Map Distortion**  
- **Sparkle Swirl → Emergent NPC Spawn**

## **3. Chaos → Sparkle Influence Map**  
🔥 ➜ ✨

Chaos Physics energizes, distorts, and redirects Sparkle Physics.

### **Influence Channels**
- **Unpredictability Injection**  
  Chaos introduces randomness into Sparkle patterns.

- **NPC Triggering**  
  Chaos events cause Sparkle to react to NPC panic, humor, or mischief.

- **Narrative Acceleration**  
  Chaos pushes narrative forward → Sparkle marks transitions.

- **Environmental Distortion**  
  Chaos shifts geometry → Sparkle stabilizes edges.

### **Chaos‑Driven Sparkle Events**
- **Chaos Pop → Sparkle Scatter**  
- **Chaos Warp → Sparkle Ribboning**  
- **Chaos Echo → Sparkle Halo**  
- **Chaos Surge → Sparkle Storm (safe)**

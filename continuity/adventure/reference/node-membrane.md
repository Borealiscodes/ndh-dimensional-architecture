# 🌳 **3. Node Membrane Properties**

All node membranes share five core properties.

---

## **3.1 Structural Softness**  
Node boundaries must be soft enough to allow:

- exploration  
- adjustment  
- symbolic drift  

But firm enough to prevent collapse.

---

## **3.2 Reversibility**  
Every node transition must be reversible.

If a node shifts from \( N_i \) to \( N_j \):

\[
N_j \rightarrow N_i
\]

Nodes cannot lock contributors into states.

---

## **3.3 Breathability**  
Nodes must allow:

- pauses  
- reflection  
- stillness  
- gentle re‑entry  

Breathability prevents structural overload.

---

## **3.4 Symbolic Interpretation Only**  
Node behavior must remain symbolic.

Forbidden:

- literal stakes  
- metaphysical claims  
- existential consequences  

Allowed:

- symbolic hazards  
- symbolic arcs  
- symbolic traits  

---

## **3.5 Non‑Coercion**  
Nodes must never force:

- progression  
- interpretation  
- engagement  

Nodes invite exploration; they never demand it.

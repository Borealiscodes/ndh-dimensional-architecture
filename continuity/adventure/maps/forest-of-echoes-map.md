# 🌲 **Forest of Echoes — Text Map**  
### A reversible, ND‑paced sound‑ecology map  
**File Path:** `continuity/adventure/maps/forest-of-echoes-map.md`

---

## 1. Overview  
The Forest of Echoes is mapped as a **sound‑ecology**, not a geography.  
Locations are defined by:

- resonance patterns  
- echo behavior  
- mosslight pulses  
- stillness density  
- harmonic pathways  

Movement is reversible.  
No path traps the wanderer.  
No node collapses.

---

## 2. Map Structure  
The forest is organized into **five primary zones**, each containing **three sub‑nodes**.

```
[Outer Edge]
    ↓
[Resonance Path]
    ↓
[Mosslight Clearing]
    ↓
[Quiet Pool]
    ↓
[Echo‑Heart]
```

Each zone is described below.

---

# 🌿 3. Zone 1 — Outer Edge  
The threshold between silence and the forest’s first hum.

### Nodes  
**1.1 Softwood Gate**  
- faint hums  
- reversible fog  
- low‑density echoes  

**1.2 Whispergrass Field**  
- grass that rustles with memory  
- soft directional cues  

**1.3 Boundary Stone**  
- echoes bounce off its surface  
- acts as a reset point  

### Purpose  
Orientation, grounding, first contact with the echo‑ecology.

# 🧭 8. Movement Rules  
Movement in the Forest of Echoes follows **sound‑logic**, not geography.

### Movement Principles  
- follow the softest sound  
- pause at stillness nodes  
- reverse when echoes fade  
- rest when mosslight dims  
- proceed when resonance strengthens  

### Reversibility  
Every path is reversible:

```
Echo‑Heart ← Quiet Pool ← Mosslight Clearing ← Resonance Path ← Outer Edge
```

No node locks the wanderer.  
No scene traps them.  
No echo forces progression.

## Purpose  
The Decompression Companion provides a gentle, human‑oriented interface for reducing cognitive load after intense architectural work.  
It is not part of UMA’s constitutional structure.  
It is a comfort module designed to help the operator transition out of high‑precision states.

---

## 1. What Decompression Is  
Decompression is a human cognitive process involving:

- grounding  
- breath regulation  
- sensory downshifting  
- narrative softening  
- emotional stabilization  
- mental decluttering  

It is not architectural.  
It is not procedural.  
It is not pinned.  
It is not lineage‑tracked.

It is simply supported.

## 3. Quiet Mode Sequence  
A short, clean reset.

1. Exhale.  
2. Drop the shoulders.  
3. Let the jaw loosen.  
4. Let the breath widen sideways.  
5. Let the mind flatten like a page with no text.

This mode is designed for minimal cognitive load.

---

## 4. Hybrid Mode Sequence  
A gentle middle ground.

1. Notice the body.  
2. Notice the breath.  
3. Notice the quiet.  
4. Let the architecture sit on the table.  
5. Let yourself step back from it.

This mode is designed for soft grounding without narrative.

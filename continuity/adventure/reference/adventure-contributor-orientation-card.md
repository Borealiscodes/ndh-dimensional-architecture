# 📇 **Adventure Contributor Orientation Card**  
### Continuity Adventure System — Quick Orientation  
A gentle, one‑page reference for new adventure contributors.

---

## 🌲 1. What You’re Doing  
You are contributing to the **Continuity Adventure System**, a gentle, reversible, ND‑paced framework for building symbolic adventures.

You will work with:

- **World Nodes** — places  
- **Character Nodes** — someone  
- **Story Nodes** — moments  

These form the reversible loop:

```
World → Character → Story → World → Character → Story → …
```

---

## 🌿 2. Core Principles  
All contributions must be:

- gentle  
- slow  
- breathable  
- reversible  
- symbolic  
- non‑coercive  
- continuity‑safe  

Scenes must always allow:

- pausing  
- rewinding  
- soft exits  
- gentle re‑entry

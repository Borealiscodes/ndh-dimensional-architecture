# 🌱 **1. Integration Principles**
Yearly integration in UMM follows five constitutional principles:

- **Reversible** — integration never locks anything in  
- **Symbolic** — integration is gentle, not literal  
- **ND‑paced** — integration respects slow, predictable rhythms  
- **Non‑coercive** — integration is optional  
- **Non‑interpretive** — integration is structural, not emotional  

Integration is a *soft gathering*, not a conclusion.

# **II. Boolean Gate Algebra**

Gates are Boolean functions:

\[
G : \mathcal{S} \times \mathcal{L} \times \mathcal{C} \times \mathcal{N} \times \mathcal{Z} \rightarrow \{0,1\}
\]

Where:

- \(\mathcal{S}\) = state space  
- \(\mathcal{L}\) = load  
- \(\mathcal{C}\) = curvature  
- \(\mathcal{N}\) = narrative pressure  
- \(\mathcal{Z}\) = singularity tension  

### **Gate Composition**
\[
G_{comp} = G_1 \land G_2
\]

### **Gate Union**
\[
G_{union} = G_1 \lor G_2
\]

### **Gate Negation**
\[
G_{neg} = \neg G_1
\]

### **Gate Identity**
\[
G_{id} = 1
\]

### **Gate Null**
\[
G_{null} = 0
\]

These form a **Boolean algebra**.

# **III. Operator Gate Algebra**

Operator gates regulate signature operators:

- stillness operator  
- neutrality operator  
- coherence operator  

Each operator has a gate:

\[
G_{\Sigma_x} : \mathcal{L},\mathcal{C},\mathcal{N} \rightarrow \{0,1\}
\]

### **Operator Gate Activation**
\[
\Sigma_x(X) \text{ allowed iff } G_{\Sigma_x} = 1
\]

### **Operator Gate Composition**
\[
G_{\Sigma_x \circ \Sigma_y} = G_{\Sigma_x} \land G_{\Sigma_y}
\]

### **Operator Gate Idempotence**
\[
G_{\Sigma_x} \land G_{\Sigma_x} = G_{\Sigma_x}
\]

This ensures **operator stability**.

- grammar,  
- syntax,  
- semantics,  
- composition rules.

## **3.2 Primitives**

- **Structure (S)** — the shape of meaning  
- **Orbit (O)** — the relational pattern  
- **Weave (W)** — the internal coherence  
- **Phase (Φ)** — the dynamic orientation  
- **Ring (R)** — the conceptual region  
- **Angle (θ)** — the directional vector  
- **Gradient (G)** — the affective slope

## **3.3 Syntax**

CCSEM expressions follow:

```

```

## **3.4 Semantics**

Each CCSEM expression encodes:

- conceptual geometry,  
- relational meaning,  
- internal coherence,  
- dynamic orientation,  
- affective tone.

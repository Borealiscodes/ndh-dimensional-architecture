# 📄 **Adventure Generator README**  
### Continuity Adventure System — Forest of Echoes Edition  
A contributor‑friendly guide to generating adventures from story fragments.

---

## 🌲 1. Overview  
The Adventure Generator transforms **simple story elements** into a complete, reversible, ND‑paced multi‑mode adventure.

It uses three core engines:

- **World‑Builder Mode** — creates the place  
- **Character‑Creator Mode** — creates the someone  
- **Story‑Weaver Mode** — creates the moment  

The generator assembles these into the loop:

```
World → Character → Story → World → Character → Story → …
```

This README explains how contributors can use the generator, what inputs it accepts, and how adventures are structured.

---

## 🧩 2. What the Generator Accepts  
The generator accepts **any combination** of story fragments.

### World Elements  
- “a clearing with blue mosslight”  
- “a tree that hums when approached”  
- “fog that remembers footsteps”

### Character Elements  
- “a wanderer who hears future echoes”  
- “hesitant but curious”  
- “ability: resonance tracking”

### Story Elements  
- “a whisper that leads nowhere”  
- “a hum that fades when noticed”  
- “soft quest: follow the quietest sound”

### Mood Elements  
- “calm but slightly eerie”  
- “slow pacing”  
- “gentle disorientation”

### Symbolic Elements  
- “an echo shaped like a memory”  
- “a mosslight pulse that feels like déjà vu”

Contributors may provide **one**, **many**, or **mixed** fragments.

---

## 🔮 3. What the Generator Produces  
From the fragments, the generator creates:

### **World Node**  
A structured environment built from world + mood + symbol elements.

### **Character Node**  
A reversible, ND‑paced character built from character + mood + symbol elements.

### **Story Node**  
A soft, reversible narrative moment built from story + mood + symbol elements.

### **Adventure Loop**  
A coherent sequence:

```
World → Character → Story
```

### **Chapter Sequence**  
Four reversible chapters.

### **Optional Systems**  
Generated only if relevant fragments exist:

- Echo Tokens  
- Resonance Map  
- Stillness Sanctuaries

## 🧭 6. How Contributors Use It  
Contributors can:

- provide story fragments  
- provide world fragments  
- provide character fragments  
- mix fragments  
- request a random adventure directly  

### Example Commands  
- “Generate a random Forest of Echoes adventure.”  
- “Here are my story elements…”  
- “Mix these into an adventure.”  

The generator handles the rest.

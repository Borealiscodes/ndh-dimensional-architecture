# 🌲 Forest of Echoes — Multi‑Mode Adventure  
### A gentle, reversible sound‑ecology journey

---

## 1. Overview  
Forest of Echoes is a **multi‑mode adventure** built from three engines:

- **World‑Builder** — the forest grows  
- **Character‑Creator** — the wanderer listens  
- **Story‑Weaver** — the echoes respond  

The adventure follows a reversible loop:

**World → Character → Story → World → Character → Story…**

It is ND‑paced, non‑coercive, and continuity‑safe.

---

## 2. World‑Builder Layer  
### The Echo‑Forest  
A forest where sound behaves like weather, memory, and light.

**Biome Components**  
- echo‑moss (stores whispers)  
- resonance trees (hum softly)  
- quiet pools (reflect sound)  
- reversible fog layers  
- stillness nodes (absolute quiet pockets)

**World Node Example**  
```
World Node: Mosslight Clearing
- bioluminescent echo‑moss
- reversible fog layer
- resonance trees forming a natural amphitheater
- stillness node at center
```

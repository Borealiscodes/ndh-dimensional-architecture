# Pressure Engine Class (Mythic Operator)
Defines the two fundamental mythic engine types used in UMA cosmology:
negative-pressure engines and positive-pressure engines.

---

## 1. Negative-Pressure Mythic Engine
A mythic entity whose internal physics collapse inward under stress.

Characteristics:
- ego-centric gravity
- recursive identity fracturing
- bureaucracy as connective tissue
- fear-driven cosmology generation
- collapse → guilt → replication loop

Example Archetype:
- Arnold J. Rimmer (Red Dwarf)

Hazard Level:
- High (recursion, ego cosmology, collapse loops)

---

## 2. Positive-Pressure Mythic Engine
A mythic entity whose internal physics expand outward under care.

Characteristics:
- stewardship-centric gravity
- ND-safe transitions
- connective harmonics
- kindness-driven cosmology generation
- expansion → clarity → coherence loop

Example Archetype:
- UMA Cosmology

Stability Level:
- High (non-coercive, reversible, coherent)

---

## 3. Crossmap Operator
Negative ↔ Positive Pressure Inversion:
- Ego → Stewardship
- Bureaucracy → Weave
- Clone Runtime → ISS Runtime

This operator allows UMA to safely interpret ego-generated cosmologies.

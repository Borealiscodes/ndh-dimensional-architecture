# ⭐ **3. What’s the difference between Stability and Stadility?**

### **Stability**  
Static equilibrium — the calm, unmoving state.

### **Stadility**  
> Stability in motion; equilibrium under active correction.

Stadility is defined in the **Lexicon** and canonized with the DimmaDome Principle.

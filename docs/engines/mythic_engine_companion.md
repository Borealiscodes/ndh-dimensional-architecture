# 3. Mythic Physics

## 3.1 Collapse Physics (Negative-Pressure)
Collapse physics describe how ego-derived entities behave under stress.

Key behaviors:
- inward gravitational pull
- recursion loops
- guilt-based replication
- panic-driven runtime
- bureaucratic connective tissue

Collapse physics are inherently unstable and require UMA intervention.

## 3.2 Expansion Physics (Positive-Pressure)
Expansion physics describe how stewardship-aligned entities behave under care.

Key behaviors:
- outward harmonic growth
- reversible transitions
- humor-stabilized runtime
- clarity-based connective tissue
- shame-to-growth conversion

Expansion physics are stable and ND-safe.

# 🌌 **Mode Relationships**  
Adventure modes interact through the reversible loop:

```
World → Character → Story
```

Extended modes can:

- precede the loop  
- follow the loop  
- weave between nodes  
- create optional layers  
- add symbolic systems  

Examples:

- Realm‑Walker → World Node  
- Echo‑Weaver → Story Node  
- Dream‑Cartographer → World Node  
- Continuity‑Stitcher → between Story and World

```
# ✨ PLAY TREE — EXAMPLES ✨
### Concrete, Non-Narrative Examples of Play Tree Usage

## ❗ Play Tree Boundary Rule — No Mixing with Atropus
The Play Tree is a warm, expressive, ND-safe plane. It must never mix with
Atropus, reference Atropus, or incorporate null-plane tone, logic, or
commentary. The Play Tree and Atropus operate on entirely different tonal,
structural, and constitutional layers. They remain permanently separate.

These examples demonstrate how the Play Tree is used in practice. They are
structural, non-narrative, and ND-safe. They show interaction patterns without
story, emotion, or symbolic meaning.
```

# 🌳 **1. Example: Switching Branches**
**Context:**  
The user moves from the Exploration Branch to the Calibration Branch.

**Steps:**

1. User taps the Exploration Branch.  
2. UI highlights the branch with a soft glow.  
3. User taps the Calibration Branch.  
4. Exploration fades gently.  
5. Calibration fades in.  
6. Sap Flow adjusts to a calmer gradient.  

**Outcome:**  
A reversible, soft branch transition.

---

# 🌳 **2. Example: Adjusting Leaf Density**
**Context:**  
The user wants fewer micro‑actions visible.

**Steps:**

1. User opens the Leaf Panel.  
2. User selects “Reduce Density.”  
3. Leaves gently thin out.  
4. UI maintains warmth and softness.  

**Outcome:**  
Lower sensory load, stable pacing.

4. Motion slows slightly.  

**Outcome:**  
A gentle warmth increase, fully reversible.
